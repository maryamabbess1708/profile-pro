## 🚀 À Propos
Ce projet est une page de profil utilisateur réalisée dans le cadre du **HTML Masterclass IRIS Junior Création**.  
Il applique les bonnes pratiques HTML5 sémantique, l'accessibilité ARIA et le workflow Git/GitHub professionnel.
---
## 🛠 Technologies Utilisées
- [HTML5](https://developer.mozilla.org/fr/docs/Web/HTML) - Structure sémantique
- [Git/GitHub](https://github.com/) - Versioning & Collaboration
---
## ✨ Fonctionnalités
- Header avec avatar (`<figure>`), nom, rôle et navigation interne (`<nav>`)
- Section **À propos** avec biographie
- Section **Compétences** avec liste structurée
- Section **Contact** avec formulaire complet (`fieldset`, `legend`, `label`)
- Footer avec copyright
- Attributs ARIA pour l'accessibilité
