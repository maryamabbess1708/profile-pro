# profile-pro
## Description
Page de profil utilisateur conçue en HTML5 sémantique et accessible,
réalisée dans le cadre du HTML Masterclass IRIS Junior Création.
Le projet respecte les standards W3C et les bonnes pratiques d'accessibilité.
## Aperçu
Page de profil avec header, sections de contenu et formulaire de contact.
## Technologies utilisées
- HTML5
- Git / GitHub
## Fonctionnalités
- Header avec avatar, nom et navigation interne
- Section "A propos" avec biographie
- Section "Competences" avec liste structurée
- Section "Contact" avec formulaire complet
- Footer avec liens réseaux sociaux
## Validation W3C
- Page validée sans erreur sur validator.w3.org
